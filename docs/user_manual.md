# Руководство пользователя (кратко)

## Роли

- **Администратор:** пользователи, аудит, синхронизация ERP.
- **Рабочий:** брак, ежедневные отчёты, просмотр схем.
- **Конструктор:** загрузка версий схем, согласование, оборудование (совместно с админом).

## Брак

Создайте заявку с описанием, цехом, приоритетом и категорией. Прикрепите фотографии (до 10 МБ). Статусы: новый → в работе → решён / отклонён.

## Схемы

Для проекта загружайте новые версии файлов (PDF или изображение). Аннотации к PDF хранятся отдельно в системе (координаты и текст).

## Отчёты

Ежедневный отчёт: дата, смена, чек-лист задач, при необходимости минуты работы и выпуск для учёта OEE.

## Даты

В интерфейсах клиентов и в экспортах используется формат **ДД.ММ.ГГГГ** там, где это реализовано на стороне отчётов.
