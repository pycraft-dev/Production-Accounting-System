# Руководство пользователя

## Вход

- Получите логин и пароль у администратора. В поле логина можно ввести **короткое имя** (например `admin`) или полный **email**.
- После первого входа с временным паролем приложение может запросить **новый пароль** (минимум 8 символов).

## Десктоп

Разделы в боковом меню: брак, схемы, отчёты, аналитика. Пункт **«Админ»** виден только администраторам.

## Мобильное приложение

Экраны: брак, отчёт, схемы, синхронизация оффлайн‑очереди. Администратор видит дополнительный пункт **«Администрирование»**.

## Адрес сервера

В настройках/переменных окружения клиента укажите URL API (например `http://192.168.1.50:8000` в локальной сети). Тот же адрес используется для проверки обновлений.
